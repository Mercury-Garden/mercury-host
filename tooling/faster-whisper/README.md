# faster-whisper — local STT for Hermes

Declarative record of the local speech-to-text install that backs Hermes'
Spanish-voice-message transcription path. Configured via
`~/.hermes/config.yaml → stt.{enabled,provider,local}` (NOT tracked here —
Hermes owns that file).

## What this host runs

- **Library:** `faster-whisper` (CTranslate2-backed Whisper reimplementation)
- **Model:** `small` (466 MB, Systran/faster-whisper-small on HF Hub)
- **Language pin:** `es` (skip auto-detect; saves ~0.5s and removes
  hallucination risk on short clips)
- **Inference:** CPU-only, aarch64 (Neoverse-N1). ~5–8s for a 30s clip.
- **Provider:** `local` (the default STT path; Groq/OpenAI/Mistral/xAI also
  supported if keys are set, but MiniMax has no STT endpoint).
- **Audio decode:** `ffmpeg` from apt (already installed — see
  `packages/apt.list`).

## Install (recovery)

The hermes-agent venv at `/home/ubuntu/.hermes/hermes-agent/venv` IS the
gateway's Python. Install there:

```bash
/home/ubuntu/.hermes/hermes-agent/venv/bin/python3 -m pip install faster-whisper
```

The `small` model downloads automatically on first STT call into
`~/.cache/huggingface/hub/models--Systran--faster-whisper-small/` (~464 MB).
Set `HF_TOKEN` env var before first call to avoid rate-limit warnings.

## Verify

```bash
/home/ubuntu/.hermes/hermes-agent/venv/bin/python3 -c "from faster_whisper import WhisperModel; print('ok')"
ls -la ~/.cache/huggingface/hub/models--Systran--faster-whisper-small/   # should exist after first STT call
```

## Config (not in this repo)

`~/.hermes/config.yaml`:

```yaml
stt:
  enabled: true
  provider: local
  local:
    model: small
    language: es
```

`stt.*` is read per-call by `tools/transcription_tools.py::transcribe_audio`
(model is lazy-loaded). The gateway reads config at request time, so config
changes take effect on the next voice message — no restart needed.

## Recoverability

**Recoverable.** No irreplaceable data is held here:

- The pip package is reinstalled via the command above.
- The model cache can be regenerated by any successful STT call (it
  downloads on demand).
- The cache lives under `~/.cache/huggingface/` which is NOT included in
  `scripts/backup-mercury-state.sh` (intentional — 464 MB of re-derivable
  data, no need to back it up).
- Hermes config (`stt.*` block) is the user's responsibility, not in this
  repo.

If the host is wiped: re-install faster-whisper, restore
`~/.hermes/config.yaml` from the secrets snapshot, send one voice message.
The model downloads itself.

## Capacity / sizing

- Model on disk: 464 MB
- RAM at inference (int8, small): ~1 GB
- Disk free required: ≥500 MB (current `/` has 6.9 GB free)
- CPU: ~5–8s per 30s clip on 4× Neoverse-N1 aarch64

If accuracy becomes a problem on accented Spanish, upgrade to `medium`
(1.5 GB, ~3× slower) or `large-v3` (3.1 GB, ~6× slower). Don't go above
`medium` for interactive chat — too slow.

## Companion deps (already installed, listed for completeness)

- `ffmpeg` — apt package, used to decode inbound Discord audio into the
  PCM format faster-whisper expects. Already in `packages/apt.list`.
- `ctranslate2`, `tokenizers`, `onnxruntime`, `huggingface-hub`, `numpy`,
  `av` — pulled in transitively by `pip install faster-whisper`. No manual
  install needed.
